# CS450

## Technical Requirements

This repository is created for the CS450 Applied Programming course.

### Branching Strategy
- Main branch: `main`
- Each homework will be developed on a separate weekly branch
- Branch naming format: `week-01`, `week-02`, ..., `week-14`

### Workflow
- Make changes for the homework
- Commit changes
- Push to the corresponding weekly branch
- Open a pull request after pushing
